# PS Restaurant Web Application

A premium, interactive digital menu and ordering platform designed for a seamless dining experience.

## 🎨 Theme & Design

The "PS Restaurant" web app features a **Modern Luxe** aesthetic designed to feel high-end and inviting:
- **Sophisticated Palette**: A deep charcoal base (`#161C24`) paired with elegant gold accents (`#E2C785`).
- **Dynamic Motion**: Features staggered entrance animations (fade-in-up) for food cards to create a "reveal" effect on load.
- **Premium UI Components**: Uses Glassmorphism (blurred backdrops) for modals and subtle shadow depth on interactive elements.
- **Micro-interactions**: Smooth scaling transitions on hover and tactile feedback on button clicks.

## ✨ Key Features

- **Interactive Digital Menu**: Browse high-quality images of food items with real-time category filtering.
- **Animated Filtering**: Instantly toggle between **Veg** and **Non-Veg** items with smooth fade-and-scale transitions.
- **Live Cart Management**: Add or remove items using tactile `+` and `-` controls. The cart persists across refreshes using `LocalStorage`.
- **Polished Payment Flow**: A multi-step checkout modal with a selection menu, cash/card handling, and success confirmation.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop using Bootstrap 5's grid system.

## 📂 Project Structure

- `index.html`: Login and welcome screen.
- `menu.html`: The main storefront and interactive menu.
- `2.html`: Shopping cart and checkout interface.
- `style.css`: Custom premium styles, animations, and transitions.
- `script.js`: Core logic for state management and UI interactions.

## 🖱️ Usage Instructions

1. **Browse & Add**: Start at `menu.html`. Hover over items to see preview effects. Click the `+` button on any item to add it to your cart.
2. **Filter Your View**: Use the dropdown filter at the top right to instantly narrowed down your choices by diet.
3. **Review Cart**: Navigate to the **Cart** page via the header. Here you can adjust quantities or remove items using the bin icon.
4. **Checkout**: Click **Place Order**. A blurred-backdrop modal will appear. Select your preferred payment method.
5. **Confirm**: Click **Pay Now**. Upon success, you will be redirected to the status page to wait for your order.
